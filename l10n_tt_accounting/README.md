# Trinidad & Tobago Accounting Localization

Safe Trinidad & Tobago accounting setup for Odoo Community with Odoo Mates Accounting.

## Key Principle

This module does not replace Odoo Accounting, Odoo Mates Accounting, or Odoo's original accounting flow. It only prepares Trinidad & Tobago defaults when selected.

## Where to use it

Go to:

Accounting → Configuration → Trinidad & Tobago → Accounting Setup

The setup is intentionally not placed in the main Odoo Settings screen to avoid disrupting Odoo or Odoo Mates settings views.

## Includes

- Trinidad & Tobago country/fiscal country setup
- TTD currency setup
- VAT 12.5%, zero-rated, exempt, and import VAT taxes
- Trinidad-oriented chart of accounts
- PAYE, NIS, and Health Surcharge payable accounts for future payroll
- Bank setup for common Trinidad & Tobago banks
- Bank journals and chart accounts
- Fiscal positions

## Safe Setup

Select only the setup options you want, then click Apply / Refresh Trinidad Setup.
